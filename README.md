# 💡 Calculo – All-in-One Smart Calculator

Calculo is a modern, responsive, and user-friendly **multi-purpose calculator web app** designed to perform everyday calculations quickly and efficiently.  

From financial calculations to health metrics — everything in one place 🚀

---

## ✨ Features

- 📈 **EMI Calculator** – Calculate monthly loan payments  
- ⚖️ **BMI Calculator** – Check your body mass index  
- 🎂 **Age Calculator** – Find your exact age instantly  
- 💸 **Discount Calculator** – Calculate savings on deals  
- 🌗 **Dark/Light Mode Toggle** (with localStorage support)  
- 📱 Fully **Responsive Design** (Mobile + Desktop)  
- ⚡ Fast, lightweight & easy to use  

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Framework:** Bootstrap 5  
- **Storage:** LocalStorage (for theme persistence)
